# 上海规则怪谈

> 你来到了夜幕下的上海，以下规则必须严格遵守。违者——后果自负。

**1. 外滩的午夜钟声**
午夜12点整，若你恰好站在外滩，听见钟声之后五分钟内切不可回头。若你听到身后传来熟悉的声音，或有湿冷的手碰你的肩膀，无论对方自称谁，绝不回应。违者，将永远成为钟声的回响。

**2. 地铁上的陌生人**
地铁运营结束后，若你仍身处车厢内，会出现一位无脸乘客。他会低声问：“你要去哪？”只能报一个真实存在的上海地名。说出不存在的地名或保持沉默，你会在车厢内看到自己的尸体。

**3. 弄堂里的猫**
深夜穿过弄堂，当你遇到三只并排蹲坐的黑猫，必须把身上的零钱全部拿出献在地上，低头走过，切勿直视猫眼。无视此举，猫眼会成为你再也无法睁开的双眸。

**4. 黄浦江的水声**
夜晚涨潮时，若有人在江边对你招手，无论其外貌如何像你重要的人，绝对不能靠近，否则你会被水草缠绕脚踝，拖入江中，永世不得超生。

**5. 浦东高楼的倒影**
进入深夜23:33到23:44之间，不要在摩天大楼的玻璃幕墙上看自己的倒影。如果不幸看到，有第三个影子出现在身后，务必立刻闭眼原地不动，数到33，否则你会成为下一个倒影的主人。

**6. 异样的出租车**
如果午夜后打到一辆车牌全是6的出租车，请不要上车。如果已经坐上，司机会问：“你喜欢上海吗？”无论你怎么回答，司机都会咧嘴笑，驶向无人区……直到你变成下一个司机。
